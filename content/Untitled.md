**Redis**

Redis는 Remote Dictionary Server의 약자로, 인메모리(In-memory) 기반의 고성능 데이터 구조 저장소이다. 주로 캐싱, 세션 관리, 실시간 분석 등 다양한 용도로 사용되며, 오픈 소스 기반의 비관계형(NoSQL) 데이터베이스 관리 시스템(DBMS)이다.
  
기본적인 flow

1. Redis 연결 정보 읽음

2. RedisTemplate을 빈으로 등록하여 Redis와 상호작용할 수 있게 한다.

3. Spring Boot 서비스에 RedisTemplate을 주입하여 Redis를 사용할 수 있게 한다.

**@Cacheable**
메서드의 실행 결과를 캐시에 저장하여, 동일 요청이 들어왔을 때 캐시된 데이터를 반환하도록 한다.﻿

- 메서드가 호출되기 전에 캐시를 먼저 확인한다.
- 캐시에 데이터가 있으면 메서드를 실행하지 않고 캐시 데이터를 반환한다.
- 캐시에 데이터가 없으면 메서드를 실행한 후 결과를 캐시에 저장한다.

@CacheEvict

메서드 호출 시 특정 키나 전체 캐시 데이터를 삭제하는 데 사용한다.

- 메서드 실행 전 또는 후에 캐시 데이터를 제거한다.
- 데이터 삽입, 수정, 삭제 등의 작업 후 기존의 잘못된 캐시 데이터를 제거하는 데 사용한다.
